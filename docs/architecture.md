## System Overview

Hermes is a full-stack algorithmic trading platform designed for backtesting and analyzing trading strategies on the Indian stock market (NSE). The system is built for high performance, capable of processing 9000+ instruments with minute-level data.

### Key Capabilities

| Capability | Description |
|------------|-------------|
| **Dual-Mode Backtesting** | Vector (fast) and Event-driven (realistic) engines |
| **Multi-Strategy Support** | RSI, MACD, Bollinger Bands, SMA Crossover, MTF |
| **High-Performance Data** | Parquet storage with in-memory caching |
| **Interactive Visualization** | Candlestick charts with signal overlays |
| **Real-World Simulation** | Slippage, commission, date filtering |
| **PostgreSQL Registry** | Metadata tracking and load logging |

## Appendix

### A. Running Locally

```bash
# Backend
cd hermes-backend
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
pip install -e ../hermes-data
uvicorn main:app --reload

# Frontend
cd hermes-frontend
npm install
npm run dev

# With Podman
podman-compose up -d
```

### B. Running Tests

```bash
# All checks (run_checks.sh)
./run_checks.sh

# Backend only
cd hermes-backend && pytest --cov=. --cov-fail-under=90

# Data layer only
cd hermes-data && pytest --cov=src/hermes_data --cov-fail-under=90

# Frontend only
cd hermes-frontend && npm run test:coverage
```
